The game isn't being especially friendly to even the most powerful computers at the moment, so we'll work on further optimizing the game to be more performant.

Save File Scares

Between sporadic instances of save file corruption and steam cloud backup's funky interaction with existing saves, we've disabled cloud saves and are working on securing your guys' precious progress.

❓ FAQ

Why is my game so laggy?
Turns out the game isn't as performant out of the box as we might've liked; but rest assured, optimizing the game further so you guys can have a smoother multiplayer experience with 5+ people is a top priority.
How can I back up my save file?
For those of you worried about losing your save data during this experimental phase of WEBFISHING, you can easily back it up by going to %appdata% and navigating from 'roaming' → 'godot' → 'app_userdata' → 'webfishing_2_new_ver' and copying the 'webfishing_data.save' file.
Can you add [insert thing here] to the game?
As much as we love your ideas, it'll be a bit before we even consider implementing things like new species or cosmetics to the game so early after its release. But it's not unlikely!
Where are the accessibility settings?
They'll be implemented sooner rather than later to accomodate as many new and differently-abled players as possible.
Can I play this on my Mac?
An OSX compatible build of WEBFISHING has been considered pretty heavily, but there are a few things that need to be worked out before porting is prioritized.
Can I work for you?
Not at this moment!
